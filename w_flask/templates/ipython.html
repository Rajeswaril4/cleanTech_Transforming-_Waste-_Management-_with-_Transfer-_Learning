Model and Development

Project Overview

This web application is powered by a deep learning model trained for image classification. The entire development process, from data preprocessing to model training and evaluation, was documented in a Jupyter Notebook (`.ipynb`).

The Model: VGG16

We utilize a pre-trained Convolutional Neural Network (CNN), specifically the VGG16 model, which was trained on the extensive ImageNet dataset. We have fine-tuned this model on a specialized dataset of municipal waste images to classify them into three categories: Biodegradable, Recyclable, and Trash. The model is saved as `vgg16.h5` and loaded by the Flask application to make live predictions.

Backend: Flask

The web server is built using Flask, a lightweight Python web framework. It handles file uploads, processes images, feeds them to the model for prediction, and renders the results on the webpage.

Frontend: HTML, CSS, JavaScript

The user interface is crafted with standard web technologies. The responsive design ensures a seamless experience across all devices, from desktops to mobile phones.
